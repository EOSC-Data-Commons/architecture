# C4 diagrams

This directory contains files for [Structurizr](https://structurizr.com/)
that were used in deliverable D3.1.

Run structurizr in this directory with:

```
docker run -d --rm -p 8080:8080 -v $PWD:/usr/local/structurizr structurizr/lite
```

And point your browser to `http://localhost:8080` to see & edit.
